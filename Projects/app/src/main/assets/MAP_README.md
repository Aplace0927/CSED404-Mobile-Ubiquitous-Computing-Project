# Map Format

```text
[Note count]
[Time (ms)] [action category]
...
```

## Action Categories

| Action ID | Action Explain                            |
|-----------|-------------------------------------------|
| `0`       | Push                                      |
| `1`       | Pull                                      |
| `2`       | Raise Up                                  |
| `3`       | Punch Down                                |
| `4`       | Bent In (Left for Right-handed hold)      |
| `5`       | Release Out (Right for Right-handed hold) |
